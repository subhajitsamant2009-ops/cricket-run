📸 Preview

A dark-themed cricket scorer with gold accents, live ball-by-ball tracking, and phone support.


✨ Features

🏏 Score runs (0–6) with a single tap
📊 Live CRR (Current Run Rate) display
🎯 2nd innings target, RRR and balls remaining
🟡 Six flash animation with gold highlight
🔵 Four highlight in teal
❌ Wicket tracking with all-out detection
🔴 Wide and No Ball extras (don't count as legal delivery)
🔄 Ball-by-ball over tracker (colour-coded dots)
📝 Live commentary log
↩️ Undo last entry (or Ctrl+Z)
🏆 Match result overlay with winner and margin
📱 Works on mobile — installable as an app

🎮 How to Score

🏟️ Match Flow

Enter team names, overs, and max wickets
Click Start Match
Score each ball using the buttons
After 1st innings — click 2nd Innings →
Chase the target — RRR updates live
Match result shows automatically
